Eclipse连接数据库步骤

1、先下载JDBC：

JDBC代表Java数据库连接(Java Database Connectivity)，它是用于Java编程语言和数据库之间的数据库无关连接的标准Java API，换句话说：JDBC是用于在Java语言编程中与数据库连接的API。

到www.mvnrepository.com

2、打开ECLIPSE,创建一个DYNAMIC WEB PROJECT项目

在lib中插入mysql-connector文件

3.打开 NEWFILE.JSP文件

插入下面代码 更改相应信息即可
